PHP 8.1

A popular general-purpose scripting language that is especially suited to web development

PostgreSQL

PostgreSQL is open-source relational database management system emphasizing extensibility and SQL compliance

MongoDB

MongoDB is a source-available cross-platform document-oriented database program

Graphql

GraphQL is an open-source data query and manipulation language for APIs, and a runtime for fulfilling queries with existing data

Redis

In-memory data structure store, used as a distributed, in-memory key–value database, cache and message broker, with optional durability.

React

React is a free and open-source front-end JavaScript library for building user interfaces based on UI components

Next.js

Next.js is an open-source web development framework enabling React-based web applications with server-side rendering and generating static websites

Redux

Redux is an open-source JavaScript library for managing and centralizing application state

React Native

React Native is an open-source UI software framework. It is used to develop applications for Android, Android TV, iOS, macOS, tvOS, Web, Windows

OpenCart

OpenCart is an online store management system.

Statamic

Statamic 3 is built as a highly extendable, standalone capable, Laravel CMS package

TypeScript

Free and open source programming language designed for the development of large applications and transpiles to JavaScript

HTML 5

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser

CSS 3

Cascading Style Sheets is a style sheet language used for describing the presentation of a document

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development
